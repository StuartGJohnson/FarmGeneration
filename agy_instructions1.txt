This is working code which uses pecan and weed assets (in the assets directory) to generate the orchard (for simulation in isaacsim). Note the current shell has access to the pxr usd library (via the conda env called usd). Also, executing env.sh adds the nvidia version of the usd library to the pythonpath. This has already been done in this shell.
